📦 安装说明

  1. 下载并解压 点击上方按钮下载 .zip 文件，解压到任意文件夹
  2. 打开扩展管理页 Chrome 地址栏输入 chrome://extensions 回车
  3. 开启开发者模式 页面右上角切换「开发者模式」开关（打开后顶部会出现工具栏）
  4. 加载扩展 点击左上角「加载已解压的扩展程序」，选择解压后的文件夹
  5. 确认安装成功 列表中会出现 CAO 卡片，图标出现在浏览器工具栏右上方
  6. 去 X 使用 打开 x.com 刷新页面，如果已在打开中则刷新即可。广告和垃圾会自动消失

🛠️ 功能详解

🤖 自动屏蔽垃圾 自动

  • 在推文详情页（x.com/xxx/status/...）自动检测每条回复
  • 检测维度：成人关键词（强 +2 / 弱 +2）、推广词（+2）、引流信号（+2）、内容混杂度（+2）、随机 handle（+1）
  • 总分 ≥ 4 触发自动屏蔽 + 隐藏，无需任何操作
  • 可在「屏蔽管理」页面关闭自动屏蔽

🚫 隐藏广告 自动

  • 在首页时间线和推文详情页自动识别并隐藏 X 的推广推文（Promoted）
  • 支持滚动加载后增量扫描
  • 可在 Popup 菜单开关

🔘 Popup 菜单 手动

  • 点击浏览器工具栏的 CAO 图标打开
  • 两个开关：隐藏广告 / 隐藏垃圾内容
  • 「屏蔽管理」按钮 → 打开管理页面（block.html）

🛡️ 手动上报 手动

  • 在推文详情页，每条回复下方均显示「上报」按钮
  • 系统漏检的垃圾回复 → 点击「上报」→ 自动屏蔽 + 弹出预填@开发者推文（点 Post 发送）
  • 仅在你认为系统没检测到的垃圾上使用

📋 关键词管理 手动

  • 在「屏蔽管理」→ 关键词板块查看/管理
  • 系统内置四大类：成人强（+2）、成人弱（+2）、推广（+2）、引流（+2）
  • 支持自定义添加关键词，可选分类和分数
  • 添加 / 删除后即时生效，无需刷新

📜 屏蔽记录与管理 手动

  • 自动记录每次屏蔽，包含账号头像、昵称、handle、屏蔽时间
  • 支持点击头像/名称跳转到 X 账号主页
  • 支持解除屏蔽（右侧按钮）
  • 支持从 X 同步屏蔽列表，跨设备管理
  • 分页浏览，每页展示 20 条

🔛 自动屏蔽开关

  • 「屏蔽管理」页面顶部的自动屏蔽开关
  • 关闭后系统仅标记垃圾（显示特征标签），不会自动屏蔽
  • 适合先观察规则是否误伤，确认后再开启

🔒 隐私说明

  • 所有数据存储在本地浏览器（chrome.storage.local），不会上传到任何服务器

  • 扩展仅操作 X/Twitter 页面 DOM，不会读取其他网站
